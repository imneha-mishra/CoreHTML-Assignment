1. <html> - Defines an Html document
  2. <head> - information for the document
  3. <title> - Defines a title for the document

  7. <small> - Defines smaller text
  8. <button> - Defines a clickable button
  9. <select> - Defines a drop-down list
  10. <form> - Defines an HTML form for user input
